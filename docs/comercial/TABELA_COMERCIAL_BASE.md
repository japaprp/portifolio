# Tabela Comercial Base por Nicho

Use esta tabela como referencia comercial inicial. Nao publicar valor fixo sem considerar diagnostico, mas usar como faixa de enquadramento rapido.

## Lanchonete Web

- Base web:
  - implatacao inicial com configuracao, perfis e fluxo principal
- Opcional mobile:
  - `Amorim PDV Mobile` para atendimento, cozinha, caixa ou dono
- Homologacao:
  - rodada obrigatoria de mesa, viagem, cozinha e fechamento
- Suporte inicial:
  - estabilizacao curta apos go-live

## Hortfruit Web

- Base web:
  - catalogo, leitura, checkout e operacao principal
- Opcional mobile:
  - `Amorim PDV Mobile` para leitura e operacao em movimento
- Homologacao:
  - rodada obrigatoria de busca, cadastro, carrinho, checkout e sync
- Suporte inicial:
  - estabilizacao curta apos go-live

## Amorim PDV Mobile

- Implantacao do app:
  - tenant, licenca, modulos, usuarios e preset operacional
- Integracao:
  - web proprio / contrato / adapter
- Homologacao:
  - login, operacao principal, sync e historico
- Suporte inicial:
  - estabilizacao curta apos liberacao real

## Regra comercial

- nunca fechar valor final sem diagnostico
- sempre separar `base`, `opcional mobile`, `homologacao` e `suporte inicial`
- quando houver site proprio do cliente, prever `adapter` ou adequacao ao contrato
