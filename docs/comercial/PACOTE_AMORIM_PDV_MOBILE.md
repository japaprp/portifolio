# Pacote Comercial: Amorim PDV Mobile

## Posicionamento

App universal para leitura, caixa, atendimento, cozinha e operacao por perfil, integrando com sistemas web proprios ou com `adapter` do cliente.

## Para quem serve

- cliente que quer mobilidade sem trocar todo o sistema
- negocio com operacao em mesa, balcao, caixa ou cozinha
- empresa que ja tem site ou ERP e quer app proprio de operacao

## O que resolve

- operacao presa no computador
- atendimento lento
- baixa visibilidade de fila e preparo
- dificuldade de leitura/cadastro em movimento
- excesso de tela para cada perfil

## Escopo base da entrega

- login por tenant, perfil e licenca
- operacao por nicho
- leitura por codigo
- comanda, mesa e viagem
- cozinha por rota
- historico, fila offline e sync
- painel ADM para liberar modulos
- onboarding de cliente

## Nichos atendidos hoje

- lanchonete
- hortfruit
- mercado
- padaria
- farmacia em base operacional com homologacao obrigatoria

## Opcionais

- integracao com `lanchonete web`
- integracao com `hortfruit web`
- integracao com `site proprio` do cliente via `food` ou `varejo adapter`
- impressora termica
- leitor bluetooth
- configuracao por perfil operacional

## Implantacao padrao

1. diagnostico do cenario do cliente
2. escolha do preset e dos modulos
3. configuracao de tenant, licenca e usuarios
4. integracao com web proprio ou adapter
5. homologacao operacional
6. liberacao de uso real

## Homologacao minima obrigatoria

- login por perfil
- busca/leitura de item
- operacao principal do nicho
- fechamento ou conclusao do fluxo
- sync e fila offline
- historico e auditoria

## Modalidade para cliente com sistema proprio

- nao adaptar o app para cada cliente
- usar `contrato food` ou `contrato varejo`
- se a API nao encaixar, usar `integration_adapter_template`
- liberar uso real so depois da homologacao

## O que nao prometer sem alinhamento

- uso em qualquer backend sem contrato
- farmácia pesada sem validar lote, validade e fluxo
- publicacao em loja sem revisar politica, build e homologacao

## Sinal verde para venda

Se o cliente quer:

- mobilidade
- operacao por perfil
- integracao controlada
- implantacao mais rapida

esse pacote ja esta em ponto forte para venda.
